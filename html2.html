Html Introduction

Comment TAG :

<!-- Not displayed in code -- >

The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers. You can use comments to explain your code, which can help you when you edit the source code at a later date.

Heading TAG :

Input:
<html >
<body >
<h1> This is heading 1 </h1>
<h2> This is heading 2 </h2>
<h3> This is heading 3 </h3>
<h4> This is heading 4 </h4>
<h5> This is heading 5 </h5>
<h6> This is heading 6 </h6>
</body >
</html >

Tip: Use h1 to h6 elements only for headings. Do not use them just to make text bold or big. Use other tags for that.

output :

This is heading 1

This is heading 2

This is heading 3

This is heading 4

This is heading 5
This is heading 6

Linking TAG :

<!Doctype Html >
<html >
<head>
<link rel="stylesheet"type"text/css" href="style.css">
</head>
<body>
<h1>alluri.ramesh</h1>
<p>This guy is very crazy</p>
</body>
</html>

Out Put:

alluri.ramesh

This guy is very crazy
